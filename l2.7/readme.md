Программа выведет переданные в функцию asChan() числа в случайном порядке.  
При каждом запуске программы, последовательность будет меняться. 
Функция merge использует select в бесконечном цикле, для чтения из двух каналов одновременно. 
Select ждёт, когда в одном из каналов появятся данные, и отправляет их в выходной канал. Когда канал закрывается, он заменяется на nil, чтобы select больше не пытался из него читать.  
После закрытия обоих входных каналов, выходной канал c закрывается, происходит return из горутины, а в мейне цикл range дочитывает все оставшиеся в канале значения и программа завершается.
